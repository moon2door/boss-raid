기존 UI를 건드리지 않는 마이크로 효과 쇼케이스

이번 버전은 레이아웃, 화면 흐름, 데이터 구조를 바꾸지 않습니다. 바꾸는 대상은 깜빡이는 텍스트의 방식, 밋밋한 카드와 칸의 내부 디테일, 선택 상태의 강조뿐입니다. Unity에서는 기존 프리팹의 자식 오브젝트를 몇 개 추가하거나 TMP/Outline/CanvasGroup 애니메이션만 얹는 방식으로 구현할 수 있습니다.

패치 원칙

  • 텍스트 자체는 사라지지 않게 하고, 브래킷/밑줄/글로우만 깜빡입니다.
  • 칸 디자인은 크기와 앵커를 유지하고 내부 선, 컬러 레일, 배지, 미터만 추가합니다.
  • 상태 시스템은 그대로 둡니다. selected, played, burger, ready, live 같은 기존 상태에 클래스만 붙인다는 전제입니다.

수정 포인트

  • INSERT COIN 텍스트 전체 점멸 대신 양쪽 마커만 점멸.
  • 팀 칸은 기존 위치 그대로, 내부 보더와 왼쪽 컬러 레일만 추가.
  • 선택 팀은 스케일 변경 없이 Outline/Glow만 펄스.

Unity

  • 마커는 별도 TMP 2개, CanvasGroup.alpha만 0.25-1 반복.
  • 내부 보더와 레일은 얇은 Image 자식으로 구현.
